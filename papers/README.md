# Papers

This repository contains three manuscripts:

1. **`coherence_gravity_coupling.tex`** — Main manuscript on coherence-modulated gravity: validation and tabletop feasibility. Explores field-dependent gravitational coupling $G_{\text{eff}}(\Phi)$ via coherence field.

2. **`null_results.tex`** — Standalone LaTeX manuscript detailing null results and exclusion limits for coherence–gravity and curvature–EM couplings. Reports systematic parameter sweeps and derives constraints on $\kappa_R$.

3. **`kappaR_to_BSM/curvature_em_to_bsm.tex`** — Framework linking curvature–EM coupling $\kappa_R$ to beyond-Standard-Model (BSM) parameter space, specifically dark photon kinetic mixing ($\varepsilon$) and axion-photon coupling ($g_{a\gamma\gamma}$).

## Build Instructions

### Main Paper (Coherence-Gravity Coupling)
```bash
cd papers
pdflatex coherence_gravity_coupling.tex
bibtex coherence_gravity_coupling
pdflatex coherence_gravity_coupling.tex
pdflatex coherence_gravity_coupling.tex
```

### Null Results Paper
```bash
cd papers
pdflatex null_results.tex
bibtex null_results
pdflatex null_results.tex
pdflatex null_results.tex
```

### BSM Parameter Space Paper
```bash
cd papers/kappaR_to_BSM
pdflatex curvature_em_to_bsm.tex
pdflatex curvature_em_to_bsm.tex  # Second pass for references
```

## Figure Generation

- **Main paper**: Figures generated by `scripts/generate_figures.py` → `papers/figures/*.pdf,*.png`
- **Null results**: Figures in `results/analysis/*.png` (auto-generated by analysis sweeps)
- **BSM paper**: Tables auto-generated by `scripts/generate_bsm_tables.py` → `papers/kappaR_to_BSM/table_*.tex`

## Data Tables

For null_results.tex:
- Consolidated tables: `results/reports/analysis_tables.tex` (LaTeX) and CSVs in `results/reports/csv/`
- To regenerate: `python scripts/generate_report.py --all`

For curvature_em_to_bsm.tex:
- BSM bounds tables: `results/bsm_bounds/*.csv`
- To regenerate: `python scripts/run_bsm_bounds.py && python scripts/generate_bsm_tables.py`
